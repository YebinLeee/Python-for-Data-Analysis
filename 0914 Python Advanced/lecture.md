## 사전(Dictionary)

- 키(key)와 값(value)을 가지는 Hash-map 구조
- 표현: `{key: value}`
```python
empty_dict = {}
d1 = {'a' : 'some_value', 'b': [1,2,3,4]}
```
- 리스트나 튜플처럼 사용 가능
```python
d1[7] = 'an integer'
# d1 = {'a' : 'some_value', 'b': [1,2,3,4], 7: 'an integer'}
d1['b'] # [1,2,3,4]
```
<br>

### 데이터 요소의 삭제
- 사전에서 키-값 쌍의 요소를 삭제하기 위해 del이나 pop() 사용

```python
d1[5] = 'some value'
del d1[5]
ret = d1.pop('b') # pop하고 삭제
```
